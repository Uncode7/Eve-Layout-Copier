EVE Layout Copier v3.8 (EVE Lacop) by Plunder Skol
--------------------------------------------------
A lightweight and secure tool to clone and manage EVE Online window layouts between characters and profiles.

ENGLISH
---------------------------------
Functions:
- Clone one character's layout to multiple others.
- Save selected source layouts into another settings profile.
- Create, delete, and switch between multiple layout profiles.
- Automatic backup before overwriting files.
- Fetches and displays character names and portraits via EVE API.
- Multilingual interface with live language switching.

Usage:
1. Select one origin character (source layout).
2. Select one or more destination characters (targets).
3. Click Execute Copy to clone the layout.
4. Use Save Origin Layout to store layouts into another profile folder.
5. Manage profiles (e.g., settings_Mining).
6. Backups are automatically saved inside the 'eve layout backup' folder.
